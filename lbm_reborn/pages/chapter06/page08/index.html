Создание восходящих коммуникаций

Для создания эффективных восходящих коммуникаций необходимо:

проводить еженедельные совещания, на которых давать слово по очереди каждому участнику;

перед составлением плана отдела запрашивать предложения у сотрудников;

проводить анонимные опросы сотрудников по важным проблемам деятельности отдела и организации в целом;

отвести специальное время для обсуждения личных вопросов для рядовых сотрудников.
