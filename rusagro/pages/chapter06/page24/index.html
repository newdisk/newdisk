После того как будет определена периодичность совещаний для каждого подразделения и для завода/компании в целом, необходимо свести их единую сетку. Эта сетка должна быть общей для заводов и управляющих компаний. Она представляет собой большую таблицу-расписание, разбитую на отдельные дни. При этом для каждого дня должны быть указаны все проводимые в этот день совещания – как по отдельным подразделениям, так и общие. Для каждого совещания должно быть указано его название, периодичность и время проведения.
