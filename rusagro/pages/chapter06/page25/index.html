Как внедрить эффективные совещания в своем подразделении? Есть последовательность шагов, которая вам поможет.

  1. Проводить собственные совещания по установленным правилам можно уже сейчас, то есть начиная с текущего момента!
  2. Следующим шагом к эффективным совещаниям должна стать разработка общей сетки совещаний для завода и управляющей компании и определение регламентов для каждого совещания. Заниматься разработкой сетки и регламентов должны специальные рабочие группы.
  3. Как только сетка будет создана, необходимо утвердить разработанные регламенты (TORs) совещаний.
  4. После этого можно следовать установленной сетке совещаний – то есть регулярно проводить совещания по графику и в соответствии с их регламентами.
  5. При этом нужно регулярно проводить оценку совещаний, чтобы повысить их эффективность.
