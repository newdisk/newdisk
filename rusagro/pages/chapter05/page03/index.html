На предприятии есть два основных вида отчетов:

Отчеты управления KPIs (KPI-отчеты)

Разовые отчеты

Отчеты управления KPIs, или KPI-отчеты, называют также постоянными/управленческими отчетами. Они нужны для оперативного реагирования на отклонения, анализа результатов подразделения/процесса и достижения установленных KPIs.

Для таких постоянных отчетов должен использоваться единый утвержденный шаблон.

Использование утвержденного шаблона обязательно и не подлежит обсуждению. На основе утвержденного шаблона каждая функция определяет набор KPIs/PIs, которые будут входить в отчет. Появляется так называемый темплейт KPI-отчета функции. Ответственность за подготовку темплейтов KPI-отчетов своей функции несет руководитель центральной функции.

Для заводов это руководители производственного управления. Готовые темплейты передаются для заполнения на заводы/подразделения операционной компании.

Разовые отчеты – это отчеты, которые создаются для анализа результатов конкретного проекта и устанавливаются только на период реализации проекта.

Разовые отчеты могут составляться в свободной форме – графической или цифровой. Форму каждого разового отчета должен определять руководитель проекта и его заказчик (внешний или внутренний).

В любом виде отчета обязательно должна быть следующая информация: план-факт-отклонение по установленному показателю!
