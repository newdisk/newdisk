Альтернативы окончательному соглашению

Если окончательное соглашение оказывается невозможным – можно постараться достичь временного решения. Например, сформулировать все пункты разногласий, поскольку в ходе переговоров точки несогласия могут оказаться неочевидными.

    Возможные варианты соглашений:

  • если в данный момент невозможно добиться договоренности по содержательной стороне вопроса – возможно, удастся договориться о процедурной стороне вопроса;

  • вместо постоянного договора можно заключить временный;

  • вместо всеобъемлющих соглашений – частичные;

  • вместо окончательного решения по всем вопросам – достичь согласия по принципиальному;

  • вместо согласия по обязательным пунктам – достичь согласия по необязательным.
