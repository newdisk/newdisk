Взаимосвязь интересов сторон: совпадающие интересы

Может сложиться ситуация, что по какому-то из параметров интересы партнеров совпадают – максимальный выигрыш для одного партнера означает максимальный выигрыш для другого. Взаимовыгодное решение очевидно – выбор альтернативы с максимальной прибылью.
