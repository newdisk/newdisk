ЗАДАНИЕ 14

Технологии защиты на основе биометрики включают:

  • распознавание пользователей по лицу

  • по отпечаткам пальцев

  • по радужной оболочке глаза

  • по голосу

  • по подписи

Для каждой технологии требуется специальное оборудование:

  • микрофон и звуковая карта – для распознавания по голосу

  • особое оборудование – для сканирования отпечатков пальцев

  • цифровая камера – для распознавания по лицу и оболочке глаза

Биометрические методы ненадежны.

  • ошибки возникают из-за неправильного использования оборудования или из-за повреждения сканируемых участков

Вы должны использовать в своих компаниях разные биометрические устройства.
